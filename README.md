# 🚀 Python Search & Extract Microservice (SaaS Core)

Легкий и быстрый микросервис для поиска в интернете и извлечения чистого контента из веб-страниц. Идеально подходит как компонент для AI-агентов, мониторинга новостей или инструментов парсинга данных.

## ✨ Основные функции

*   **Free Web Search**: Поиск в реальном времени через DuckDuckGo (без API-ключей).
*   **Content Extraction**: Автоматическая очистка страниц от рекламы, меню и футеров с помощью `trafilatura`.
*   **Ready for AI**: Возвращает чистый текст, готовый для передачи в LLM (ChatGPT, Claude и др.).
*   **Dockerized**: Полная готовность к развертыванию в облаке через Docker Compose.
*   **Self-documenting**: Интерактивная документация API через Swagger/OpenAPI.

## 🛠 Технологический стек

*   **Язык**: Python 3.10+
*   **Фреймворк**: FastAPI (асинхронный веб-сервер)
*   **Поиск**: `duckduckgo-search`
*   **Парсинг**: `trafilatura`
*   **Контейнеризация**: Docker & Docker Compose

📡 API Эндпоинты
1. Поиск (GET /search)
Ищет информацию и возвращает список ссылок с кратким описанием.
Параметр q: Поисковый запрос.
Пример: http://localhost:8000/search?q=новости+python+2024
2. Извлечение текста (GET /extract)
Скачивает страницу по ссылке и вырезает только полезный текст.
Параметр url: Ссылка на статью.
Пример: http://localhost:8000/extract?url=https://example.com
